üí∞ Tema Besar: Pengelolaan Keuangan Sederhana

Proyek mid-semester ini wajib bertema finansial, yaitu bagaimana menggunakan Python untuk membantu mengelola uang, transaksi, atau laporan keuangan sederhana.

  • Menggunakan list dan dictionary untuk data transaksi.
  • Memiliki menu utama dan beberapa fitur keuangan.
  • Mampu menampilkan ringkasan / laporan keuangan sederhana.

üîÄ Flowchart atau Pseudocode?

Kamu boleh memilih: merancang alur program dengan flowchart atau pseudocode.

Pseudocode

Pseudocode menuliskan langkah-langkah dengan bahasa campuran Indonesia‚ÄìInggris yang mirip kode, tapi tidak harus mengikuti sintaks Python 100%. 

BEGIN
    WHILE True:
        Tampilkan menu
        Input pilihan

        IF pilihan == 1 THEN
            proses_transaksi()
        ELSE IF pilihan == 2 THEN
            tampilkan_laporan()
        ELSE IF pilihan == 3 THEN
            BREAK
        ELSE
            Tampilkan "Input tidak valid"
    END WHILE
END

üßæ Menulis Pseudocode Alur Program

Pastikan struktur program keuanganmu sudah terlihat jelas sebelum diubah ke Python.

Kata Kunci yang Disarankan

  • BEGIN / END
  • INPUT, OUTPUT
  • IF / ELSE IF / ELSE
  • WHILE, FOR EACH
  • CALL (untuk memanggil fungsi)

Fokus pada alur logika, bukan detail sintaks Python.

Kerangka Pseudocode Utama

MAIN PROGRAM:
    Inisialisasi data keuangan (list, dictionary)
    WHILE True:
        Tampilkan menu
        INPUT pilihan user

        IF pilihan == 1 THEN
            CALL fitur_1_mencatat_transaksi()
        ELSE IF pilihan == 2 THEN
            CALL fitur_2_menampilkan_ringkasan()
        ELSE IF pilihan == 3 THEN
            BREAK
        ELSE
            OUTPUT "Pilihan tidak valid"
    END WHILE

Tugas: di worksheet, lengkapi kerangka ini dengan nama fitur yang sesuai dengan ide aplikasi keuanganmu.
