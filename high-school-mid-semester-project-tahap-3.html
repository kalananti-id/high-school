üß© Mengisi Logika Fitur Utama

Sekarang saatnya mengubah rencana di Tahap 1 & 2 menjadi kode yang benar-benar bekerja.

Dari Rencana ke Kode

  • Tinjau kembali daftar fitur di worksheet Tahap 3.
  • Pilih dulu fitur inti: misalnya catat transaksi dan ringkasan keuangan.
  • Isi bagian dalam fungsi sesuai langkah-langkah yang sudah kamu tulis.
  • Gunakan komentar untuk menandai langkah penting dalam fungsi.

Contoh ‚Äì Fitur Ringkasan

def tampilkan_ringkasan(transaksi):
    print("=== Ringkasan Keuangan ===")
    total_pemasukan = 0
    total_pengeluaran = 0

    for t in transaksi:
        if t["jenis"] == "pemasukan":
            total_pemasukan += t["nominal"]
        elif t["jenis"] == "pengeluaran":
            total_pengeluaran += t["nominal"]

    saldo = total_pemasukan - total_pengeluaran

    print("Total pemasukan :", total_pemasukan)
    print("Total pengeluaran:", total_pengeluaran)
    print("Saldo akhir      :", saldo, "\n")

üêû Debugging: Mengenali Error

Di Tahap 3, error itu wajar. Yang penting: kamu bisa membaca, memahami, dan memperbaikinya.

Jenis Error yang Sering Muncul

  • SyntaxError ‚Äì salah penulisan (koma, titik dua, kurung).
  • IndentationError ‚Äì spasi/tab tidak konsisten.
  • NameError ‚Äì nama variabel/fungsi belum didefinisikan.
  • ValueError ‚Äì input tidak cocok dengan tipe data.
  • Logical Error ‚Äì program jalan, tapi hasilnya salah.

Strategi Dasar Debugging

  • Baca pesan error pelan-pelan, perhatikan baris kodenya.
  • Cek kembali nama variabel & fungsi (harus konsisten).
  • Gunakan print() untuk melihat isi variabel.
  • Jalankan program sedikit-sedikit, bukan semuanya sekaligus.
  • Catat error di worksheet: masalah ‚Äì penyebab ‚Äì perbaikan.

üõ†Ô∏è Debugging: Contoh Masalah & Solusi

Gunakan pola Error ‚Äì Penyebab ‚Äì Solusi seperti di worksheet untuk mendokumentasikan perbaikanmu.

Contoh 1

Penyebab: Fungsi tampilkan_ringkasan() dipanggil sebelum didefinisikan.

Solusi: Pindahkan definisi fungsi ke atas, sebelum bagian while True, atau cek kembali urutan cell di Colab.

Contoh 2

Penyebab: Salah key dictionary ("nomial" bukan "nominal").

Solusi: Perbaiki semua penulisan key menjadi "nominal" dan jalankan ulang perulangan.

Aktivitas di Kelas

2. Catat minimal 2 error yang muncul di worksheet:

  • Masalah/Error

  • Cara memperbaiki

Dokumentasi debugging adalah bagian penting dari kerja seorang programmer.
