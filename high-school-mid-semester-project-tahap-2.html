üß© Membangun Fungsi-Fungsi Inti

Fungsi membantu memecah program menjadi bagian-bagian kecil yang bisa dipakai ulang dan lebih mudah dibaca.

Contoh Fungsi ‚Äì Catat Transaksi

def catat_transaksi(transaksi):
    print("=== Catat Transaksi ===")
    tanggal = input("Tanggal (YYYY-MM-DD): ")
    jenis = input("Jenis (pemasukan/pengeluaran): ")
    kategori = input("Kategori: ")
    nominal = int(input("Nominal: "))

    data = {
        "tanggal": tanggal,
        "jenis": jenis,
        "kategori": kategori,
        "nominal": nominal
    }
    transaksi.append(data)
    print("Transaksi berhasil dicatat!\n")

Tugas di Kelas

  • Buat minimal 2 fungsi inti yang sesuai temasimu.
  • Misalnya:
    • catat_transaksi()
    • tampilkan_ringkasan()
    • tampilkan_riwayat() (opsional)
  • Pastikan komentar (docstring) menjelaskan tujuan fungsi.

üìú Merancang Menu Utama

Menu utama adalah ‚Äúpintu masuk‚Äù ke semua fitur aplikasi keuanganmu.

Contoh Menu Utama

def tampilkan_menu():
    print("=== Aplikasi Keuangan ===")
    print("1. Catat Transaksi")
    print("2. Lihat Ringkasan")
    print("3. Keluar")

while True:
    tampilkan_menu()
    pilihan = input("Pilih menu (1-3): ")

    if pilihan == "1":
        catat_transaksi(transaksi)
    elif pilihan == "2":
        tampilkan_ringkasan(transaksi)
    elif pilihan == "3":
        print("Terima kasih sudah menggunakan aplikasi.")
        break
    else:
        print("Pilihan tidak valid, coba lagi.\n")

Hal yang Perlu Dicek

  • Menu tampil jelas dan mudah dipahami.
  • Program tidak berhenti kecuali user memilih keluar.
  • Setiap pilihan menu memanggil fungsi yang tepat.
  • Pesan error sopan ketika input tidak valid.

üêû Mengatasi Error Dasar di Tahap 2

Saat membuat skeleton code, error itu wajar. Yang penting kamu tahu cara membaca dan memperbaikinya.

Jenis Error yang Sering Muncul

  • IndentationError: spasi/tab tidak rapi.
  • NameError: fungsi/variabel dipanggil sebelum didefinisikan.
  • SyntaxError: tanda titik dua, kurung, atau kutip kurang.
  • ValueError: input tidak cocok dengan tipe data yang diminta.

Strategi Debugging Sederhana

  • Baca pesan error pelan-pelan, lihat baris ke berapa.
  • Gunakan print() untuk mengecek nilai variabel.
  • Jalankan kode sedikit demi sedikit, bukan semuanya sekaligus.
  • Minta teman cek struktur kode (pair-check).
